

## 在当前页面的.json 文件中引用

```javascript 
{
  "usingComponents": {
    "DefaultPage": "../../components/default-page/default-page"
  }
}
```



### 参数

- type: Number  必传
```
1 加载中 
2 暂无数据 
3 404         
4 搜索无结果 
5 下拉刷新
6 暂无评论  
7 没有更多了
8 无网络 
9 加载失败
```

- fullScreen: Boolean 不必传

- vertical: String 不必传
